# 🧠 WhiteSync - Real-Time Collaborative Whiteboard

A real-time collaborative whiteboard application built with modern web technologies.

Users can create or join rooms and draw together in real-time with live cursor tracking.

---

## 🚀 Features

- 🎨 Pencil, Line, Rectangle tools
- 🔄 Undo / Redo
- 🗑 Clear Canvas
- 🔑 Room-based collaboration
- 📋 Copy Room ID
- 🚪 Leave Room
- 🖱 Live Cursor Tracking
- ⚡ Real-time sync using Socket.IO
- 💻 Clean SaaS-style UI

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- CSS
- Socket.IO Client

### Backend
- Node.js
- Express
- Socket.IO
- CORS
